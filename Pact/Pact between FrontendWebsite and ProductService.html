A pact between FrontendWebsite and ProductService

Requests from FrontendWebsite to ProductService

  • A request to get all products with data given products exist

  • A request to get all products without auth token given products exist

  • A request to get all products without data given no products exist

  • A request to get product with ID 10 given products exist

  • A request to get product with ID 10 given no products exist

  • A request to get product with ID 10 without auth token given products exist

Interactions

Given products exist, upon receiving a request to get all products with data from FrontendWebsite, with

{
  "method": "GET",
  "path": "/products",
  "headers": {
    "Authorization": "Bearer 2019-01-14T11:34:18.045Z"
  }
}

ProductService will respond with:

{
  "status": 200,
  "headers": {
    "Content-Type": "application/json; charset=utf-8"
  },
  "body": [
    {
      "id": "09",
      "name": "Gem Visa",
      "type": "CREDIT_CARD"
    },
    {
      "id": "10",
      "name": "28 Degrees",
      "type": "CREDIT_CARD"
    },
    {
      "id": "11",
      "name": "MyFlexiPay",
      "type": "PERSONAL_LOAN"
    }
  ]
}

Given products exist, upon receiving a request to get all products without auth token from FrontendWebsite, with

{
  "method": "GET",
  "path": "/products"
}

ProductService will respond with:

{
  "status": 401
}

Given no products exist, upon receiving a request to get all products without data from FrontendWebsite, with

{
  "method": "GET",
  "path": "/products",
  "headers": {
    "Authorization": "Bearer 2019-01-14T11:34:18.045Z"
  }
}

ProductService will respond with:

{
  "status": 200,
  "headers": {
    "Content-Type": "application/json; charset=utf-8"
  },
  "body": [

  ]
}

Given products exist, upon receiving a request to get product with ID 10 from FrontendWebsite, with

{
  "method": "GET",
  "path": "/product/10",
  "headers": {
    "Authorization": "Bearer 2019-01-14T11:34:18.045Z"
  }
}

ProductService will respond with:

{
  "status": 200,
  "headers": {
    "Content-Type": "application/json; charset=utf-8"
  },
  "body": {
    "id": "10",
    "name": "28 Degrees",
    "type": "CREDIT_CARD"
  }
}

Given no products exist, upon receiving a request to get product with ID 10 from FrontendWebsite, with

{
  "method": "GET",
  "path": "/product/10",
  "headers": {
    "Authorization": "Bearer 2019-01-14T11:34:18.045Z"
  }
}

ProductService will respond with:

{
  "status": 404
}

Given products exist, upon receiving a request to get product with ID 10 without auth token from FrontendWebsite, with

{
  "method": "GET",
  "path": "/product/10"
}

ProductService will respond with:

{
  "status": 401
}
